# Coin Tapper - Telegram Mini App

A fun and engaging Telegram mini app where users can tap coins to earn points and compete with friends!

## Features
- 🎯 **Tap Coins** - Click coins rapidly to earn points
- 💰 **Earn Rewards** - Accumulate coins and unlock special rewards
- 🏆 **Leaderboard** - Compete with friends and other players
- ⚡ **Real-time Updates** - Instant feedback on your taps
- 📱 **Telegram Integrated** - Seamless Telegram mini app experience

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Telegram account

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/r84050/coin-tapper.git
   cd coin-tapper
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open your browser to `http://localhost:3000`

1. Tap the coin as fast as you can
2. Earn points for each successful tap
3. Watch your score increase in real-time
4. Invite friends to compete against you
5. Climb the leaderboard!
